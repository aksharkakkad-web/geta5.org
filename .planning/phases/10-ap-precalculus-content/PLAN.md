### UNIT 1: Polynomial and Rational Functions

**MCQ Plan: 65 questions**
- Easy (13): Basic function definitions, domain/range, simple polynomial identification, basic transformation recognition
- Medium (29): Average rate of change calculations from tables/graphs, zero identification, end behavior, transformation combinations, vertical asymptotes, holes, rational function domain
- Hard (23): Complex zeros (conjugate pairs), multiplicity analysis from graphs, polynomial long division, rational function graphs with multiple features, piecewise models, combining transformations

**Topic Allocation:**
- 1.1–1.3 (Covariation, Rates of Change): 8 questions
- 1.4–1.6 (Polynomial behavior): 14 questions
- 1.7–1.11 (Rational functions): 18 questions
- 1.12 (Transformations): 12 questions
- 1.13–1.14 (Function models): 13 questions

**Calculator/No-Calculator Notes:**
- No-calculator emphasis: exact algebraic manipulation of polynomial/rational expressions, exact zeros, end behavior determination
- Calculator: graphical interpretation questions, model selection from data

**Stimulus Plan:**
- `chart`: ~30 questions — graphs of polynomials/rational functions, concavity, transformation graphs
- `table`: ~18 questions — rate of change tables, sequence tables, function value tables
- `text`: ~1 question — contextual scenarios (crop yield, light intensity)
- `none`: ~16 questions — pure symbolic/algebraic questions

**Drill Plan (name_to_formula and definition_to_term):**

*name_to_formula (~20 cards):*
- General polynomial form
- Average rate of change formula
- Vertical translation formula
- Horizontal translation formula
- Vertical dilation formula
- Horizontal dilation formula
- Polynomial long division form
- Even function property
- Odd function property
- Secant line slope formula
- Binomial theorem (n=2)
- Remainder theorem
- Standard form vs factored form relationship
- Rational function end behavior (equal degrees)
- Horizontal asymptote (denom dominates)

*definition_to_term (~25 cards):*
- Function maps each input to exactly one output → function
- Set of all input values → domain
- Set of all output values → range
- Rate at which output changes per unit input → rate of change (or slope)
- Graph bends upward (rate of change increasing) → concave up
- Graph bends downward (rate of change decreasing) → concave down
- Input where output = 0 → zero (root)
- Where function switches from increasing to decreasing → local maximum
- Where function switches from decreasing to increasing → local minimum
- Highest/lowest value on entire domain → global (absolute) maximum/minimum
- Where rate of change switches from increasing to decreasing → inflection point
- Non-real zeros come in pairs (a+bi, a-bi) → complex conjugate pairs
- Number of times a factor (x-a) appears → multiplicity
- Degree n polynomial has exactly this many complex zeros → n zeros (counting multiplicity)
- Function where f(-x)=f(x) → even function
- Function where f(-x)=-f(x) → odd function
- Output approaches b as x→∞ → horizontal asymptote y=b
- Graph increases without bound at x=a → vertical asymptote x=a
- Point where rational function is undefined but limit exists → hole
- Quotient polynomial when numerator degree = denominator degree + 1 → slant asymptote
- Different behavior rules over different domain intervals → piecewise-defined function
- Coefficient of highest-degree term → leading coefficient
- Highest power in a polynomial → degree

### UNIT 2: Exponential and Logarithmic Functions

**MCQ Plan: 60 questions**
- Easy (12): Basic exponential growth/decay identification, simple log evaluation, arithmetic vs geometric sequence identification, basic composition
- Medium (27): Exponential function manipulation (equivalent forms), log properties application, solving simple exponential equations, inverse function identification, data modeling context
- Hard (21): Complex exponential equation solving, log manipulation with multiple properties, composition of inverse functions, semi-log plot interpretation, model validation, inverse function construction

**Topic Allocation:**
- 2.1–2.2 (Sequences, linear vs exponential): 10 questions
- 2.3–2.5 (Exponential functions, manipulation, modeling): 15 questions
- 2.6–2.8 (Model validation, composition, inverse): 10 questions
- 2.9–2.12 (Logarithms, manipulation): 15 questions
- 2.13–2.15 (Equations, modeling, semi-log): 10 questions

**Calculator/No-Calculator Notes:**
- No-calculator: exact logarithm evaluation, algebraic manipulation of exponential/log expressions, exact solutions
- Calculator: regression modeling, semi-log interpretation, data modeling

**Stimulus Plan:**
- `table`: ~25 questions — sequence tables, function value tables, rate of change tables
- `chart`: ~17 questions — graphs of exponential/logarithmic functions, semi-log plots
- `text`: ~3 questions — contextual problems (population growth, radioactive decay, finance)
- `none`: ~15 questions — pure algebraic/symbolic questions

**Drill Plan (name_to_formula and definition_to_term):**

*name_to_formula (~20 cards):*
- Arithmetic sequence general term
- Geometric sequence general term
- Exponential function general form
- Compound interest formula
- Continuous growth formula
- Product property of exponents
- Power property of exponents
- Logarithm definition (b^y = x)
- Product rule of logarithms
- Quotient rule of logarithms
- Power rule of logarithms
- Change of base formula
- Natural log as log base e
- Inverse relationship: b^{log_b(x)} = x
- Inverse relationship: log_b(b^x) = x
- Linear function form (slope-intercept)
- Composition of functions

*definition_to_term (~20 cards):*
- Successive terms have a common difference → arithmetic sequence
- Successive terms have a common ratio → constant proportional change → geometric sequence
- Output values change proportionally over equal-length intervals → exponential function
- Output values change at constant rate over equal-length intervals → linear function
- Function applied to the output of another function → composition
- Function that reverses another function (switches domain/range) → inverse function
- log_b(x) = y means → b^y = x (definition of logarithm)
- log base e → natural logarithm (ln)
- Logarithm of a product = sum of logarithms → product rule
- Logarithm of a quotient = difference of logarithms → quotient rule
- Logarithm of a power = coefficient times logarithm → power rule
- Plot where y-axis is logarithmically scaled → semi-log plot
- On a semi-log plot, exponential data appears as this → straight line
- f(g(f^{-1}(x))) simplifies because of this → inverse function property
- When a>0 and b>1 in f(x)=ab^x → exponential growth
- When a>0 and 0<b<1 in f(x)=ab^x → exponential decay

**Calculator/No-Calculator Notes:**

**Stimulus Plan:**

- `table`: ~15 questions — function value tables, rate of change tables for polar functions
- `text`: ~3 questions — contextual periodic phenomena (electricity usage, tides, temperature)
- `none`: ~17 questions — exact value questions, identity manipulation

**Drill Plan (name_to_formula and definition_to_term):**

*name_to_formula (~25 cards):*
- Pythagorean identity
- sin^2 + cos^2 = 1 alternate form (sec^2)
- sin^2 + cos^2 = 1 alternate form (csc^2)
- Sine sum identity
- Cosine sum identity
- Double-angle for sine
- Double-angle for cosine
- tan(θ) definition
- sec(θ) definition
- csc(θ) definition
- cot(θ) definition
- Sinusoidal function general form
- Amplitude formula
- Period formula (sinusoidal)
- Phase shift formula
- Period of tangent
- x = rcos(θ) conversion
- y = rsin(θ) conversion
- r = sqrt(x^2 + y^2) conversion
- θ = arctan(y/x) conversion
- Point on circle radius r: P = (rcos, rsin)
- Midline formula
- Exact value: sin(π/6)
- Exact value: cos(π/3)
- Exact value: sin(π/4)

*definition_to_term (~25 cards):*
- Smallest positive period k where f(x+k)=f(x) → period
- Half the difference between max and min output values → amplitude
- Horizontal shift of sinusoidal function → phase shift
- y=d in a sinusoidal function, average of max and min → midline
- Function that repeats output values at regular intervals → periodic function
- Angle measure based on arc length to radius ratio → radian
- y-coordinate of point where terminal ray meets unit circle → sine
- x-coordinate of point where terminal ray meets unit circle → cosine
- Slope of terminal ray on unit circle → tangent
- Reciprocal of cosine → secant
- Reciprocal of sine → cosecant
- Reciprocal of tangent → cotangent
- Restriction of sine to [-π/2, π/2] for invertibility → domain restriction for arcsin
- Restriction of cosine to [0, π] for invertibility → domain restriction for arccos
- arcsin, arccos, arctan → inverse trigonometric functions
- Coordinate system using (r, θ) pairs → polar coordinate system
- Distance from origin in polar coordinates → r (radius)
- Angle from positive x-axis in polar coordinates → θ (angle/argument)
- Graph of r = f(θ) in polar coordinate system → polar function graph
- Rate of change of radius r per radian of angle θ → average rate of change of polar function

## Study Guide Plans

### Unit 1 Study Guide
- **Theme:** Modeling Change: Polynomial and Rational Functions
- **Core concepts (5–8):** covariation, rates of change, polynomial zeros/multiplicity, end behavior, rational function asymptotes/holes, transformations, function model selection
- **Formulas (8–10):** general polynomial form, average rate of change, vertical/horizontal translation/dilation, even/odd function, polynomial long division, horizontal asymptote rule
- **Exam tip:** Focus on reading graphs for concavity (rate of change increasing/decreasing)

### Unit 2 Study Guide
- **Theme:** Exponential Growth and Logarithmic Inversion
- **Core concepts (5–8):** arithmetic vs geometric sequences, linear vs exponential change, exponential function properties, log as inverse of exponential, log properties, composition and inverse functions, semi-log plots
- **Formulas (8–10):** exponential function, arithmetic/geometric sequence, product/quotient/power log rules, change of base, inverse relationship
- **Exam tip:** Master the log rules cold — most algebraic manipulation on Part B FRQ requires log properties without a calculator

### Unit 3 Study Guide
- **Theme:** Circular Motion and Periodic Patterns: Trigonometric and Polar Functions
- **Core concepts (5–8):** radian measure, unit circle, sinusoidal transformations, trig identities, inverse trig restrictions, polar coordinates, polar function rates
- **Formulas (10–12):** Pythagorean identity, sin/cos sum identities, double-angle formulas, sinusoidal general form, amplitude/period, tan/sec/csc/cot definitions, polar conversion formulas
- **Exam tip:** Memorize the unit circle for multiples of π/6, π/4, π/3 — no calculator on Part B
