## Notes
- CED = Course and Exam Description — canonical topic list, formula list, and skill categories
- Do NOT attempt to convert this PDF to markdown — formula tables do not survive conversion cleanly
- The Researcher agent will read the PDF directly via the Read tool
- AP Precalculus is relatively new (first exam 2024) — prioritize the most recent CED
