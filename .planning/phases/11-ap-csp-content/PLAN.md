## Stimulus Plan Per Unit

### Unit 1 (Creative Development) — 55 MCQs
- ~85% must have stimulus → ~47 with stimulus, ~8 none
- Stimulus types: text passages about computing innovations, collaboration scenarios, debugging scenarios
- No code stimuli (Unit 1 is concept-heavy)
- Multi-select: 2 questions (error types, iterative vs. incremental)

### Unit 2 (Data) — 60 MCQs
- ~85% stimulus → ~51 with stimulus, ~9 none
- Stimulus types: code (binary/data operations ~30%), text (~55%), table (~15%)
- Code stimuli: binary operations, data manipulation, metadata extraction
- Multi-select: 2 questions

### Unit 3 (Algorithms & Programming) — 90 MCQs
- ~85% stimulus → ~77 with stimulus, ~13 none
- Stimulus types: code (~75% of stimuli), text (~25% of stimuli)
- ALL code stimuli use CB pseudocode exclusively
- Multi-select: 4 questions (Boolean logic, algorithm selection, efficiency)

### Unit 4 (Computer Systems & Networks) — 55 MCQs
- ~85% stimulus → ~47 with stimulus, ~8 none
- Stimulus types: text passages (network scenarios, fault tolerance), table (protocol comparison), no code
- Multi-select: 2 questions

### Unit 5 (Impact of Computing) — 65 MCQs
- ~85% stimulus → ~55 with stimulus, ~10 none
- Stimulus types: text passages (innovation scenarios, ethics cases), table (data about digital divide)
- Minimal code
- Multi-select: 3 questions
