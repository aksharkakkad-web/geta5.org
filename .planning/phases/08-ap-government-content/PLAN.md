- `type: "table"` — polling data, election results, congressional vote tallies, demographic breakdowns
- `type: "chart"` — Chart.js line/bar for party identification trends, approval ratings, ideological distribution

## Study Guide Structure

Each unit produces one study guide JSON file following the `study-guide.schema.json` format:
- `theme` — concise unit title/theme
- `core_concepts` — array of concept strings (plain English + any relevant text in quotes)
- `key_terms` — derived from `is_key_term: true` drill cards at render time (not stored in study guide JSON)
- `formulas` — empty array `[]` for AP Government (no math formulas; no `description`/`example` fields needed since this is not a math subject)
- `exam_tip` — concise strategy note for AP exam

Study guide IDs: `gov-sg-unit-1`, `gov-sg-unit-2`, `gov-sg-unit-4`
